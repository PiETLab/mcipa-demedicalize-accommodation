

## Steps
1. In `GitHub Desktop`, choose menu option `Repository` → `Open in Terminal`.
2. In that terminal, run
```bash
gh auth login
```
3. You will next see a command-line interface.  For the question `Where do you use GitHub?`, choose the answer `GitHub.com`
4. For the question `**What is your preferred protocol for Git operations on this host?**`, answer `HTTPS`
5. 
   **Authenticate Git with your GitHub credentials?** Y
6. For question `**How would you like to authenticate GitHub CLI?**`, answer `**Login with a web browser**`
7. Follow the instructions, Press Enter to open the browser
8. In the browser, 
   ![[IMG-20260702191220411.png|Pasted image 20260702191220.png]]
   Select `Continue`
9. Use the code in the Terminal and click `Continue`
   ![[IMG-20260702191243447.png|Pasted image 20260702191243.png]]
10. You will see something like this (won't be this exactly)
    ![[IMG-20260702191350152.png|Pasted image 20260702191350.png]]



	Click `Authorize github`

11. There will be one more step to authorize access.  complete this step
12. You can now close the terminal.
