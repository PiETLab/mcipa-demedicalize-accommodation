# Code of Conduct

> **Placeholder** — this document will be expanded before the project moves beyond prototype phase.

## Our pledge

We are committed to making participation in this project a respectful and harassment-free experience for everyone.

## Expected behaviour

- Be respectful and constructive in all interactions.
- Assume good intent; ask for clarification before assuming bad faith.
- Focus feedback on content and ideas, not people.

## Unacceptable behaviour

- Harassment, intimidation, or discrimination of any kind.
- Personal attacks or sustained disruption of discussion.

## Reporting

To report a conduct concern, please contact the repository maintainer directly via GitHub ([@mbaljko](https://github.com/mbaljko)) or open a private issue.
