## Reporting a vulnerability

This repository is a static site generator setup with no user-facing authentication, databases, or sensitive data processing.

If you discover a potential security issue (e.g. exposed credentials, a dependency with a known CVE), please:

1. **Do not open a public issue.**
2. Contact the maintainer directly via GitHub: [@mbaljko](https://github.com/mbaljko).
3. Include a brief description of the issue and steps to reproduce.

We will acknowledge your report within 5 business days and aim to address it promptly.
